
You need to create a folder in the training class and upload the files you want to train in this folder. Audio files with the extension "an36-fejs-b.flac" flac structure
required. If you use a different structure, you will get an error. If you want to test after completing the training, create a test folder, upload your own voice into it and test the test.
you can do it. The extension of the audio file you want to test must also have a flac structure.


Egitim sınıfında bir klasör oluşturup bu klasörün içine eğitime almak istediğiniz dosyalarını yüklemeniz gerekmektedir. Ses dosyalarının uzantısı "an36-fejs-b.flac" flac yapısı olması 
gerekmektedir. Farklı bir yapı kullanırsanız hata alırsınız. Eğitimi tamamladıktan sonra test yapmak istiyorsanız bir test klasörü oluşturup onun içine kendi sesinizi yükleyip testi 
yaptırabilirsiniz. Test yaptırmak istediğiniz ses dosyasının uzantısı da flac yapısı olması gerekmektedir.
